# Fast Food CLI

A simple command line fast food ordering program written in C.

The program allows the user to choose a food item, enter the quantity and calculates the total price.

## Features

- Choose between different food items
- Display item price
- Enter quantity
- Calculate total price
- Simple command line interface

## How It Works

1. The program asks the user which food item they want
2. The user selects between:
   - pizza
   - burger
   - sendvic
3. The program displays the price of the selected item
4. The user enters the quantity
5. The total price is calculated and displayed

- C programming language
- Standard C libraries
